# Superhero Battle Arena 

This is a console-based C# project demonstrating Object-Oriented Programming concepts.

## Heroes

- **Warrior**: A strong melee fighter with a sword.
- **Mage**: A magic user that casts fire spells.
- **Archer**: A ranged fighter using arrows.

## Battle System

- Heroes attack each other using the `Attack()` method.
- Each attack reduces the enemy's health using arithmetic operators.
- The game checks if a hero is alive after each attack.
- Heroes can heal using:
  - `Heal()` → heals 20 HP
  - `Heal(int amount)` → heals a custom amount
